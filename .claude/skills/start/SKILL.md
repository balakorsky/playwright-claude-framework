# Skill: start

Инициализация рабочей сессии. Вызывается командой `/start`.

## Шаги выполнения

1. **Прочитай SNAPSHOT.md** — `cat .claude/SNAPSHOT.md`
   - Покажи секцию `## Последняя сессия` — когда была и что делали
   - Покажи секцию `## Приоритеты` — что запланировано на сейчас
   - Покажи секцию `## Известные проблемы` если она не пустая

2. **Git state** — выполни `git status` и `git log --oneline -5`
   - Если есть незакоммиченные изменения — сообщи какие файлы изменены
   - Если ветка отстаёт от remote — предупреди

3. **Проверь stash** — `git stash list`
   - Если есть stash — сообщи пользователю

4. **Посчитай тесты** — `npx playwright test --list 2>/dev/null | grep "Total:"`:
   - api, chromium, e2e, visual, a11y

5. **Сводка** — выведи отчёт:

```
=== Сессия начата ===
Последняя сессия: <дата из SNAPSHOT.md>
Тогда делали: <краткое из SNAPSHOT.md>

Приоритеты сейчас:
  1. <из SNAPSHOT.md>
  2. ...

Ветка: main (актуальна / отстаёт на N коммитов)
Незакоммиченных файлов: N
Последний коммит: <сообщение>

Тесты:
  api    — N  |  e2e  — N
  ui     — N  |  visual — N  |  a11y — N

Готов к работе. Продолжаем?
```

## Правила

- Не запускай тесты — только `--list`
- Не делай коммиты и пуши
- Если SNAPSHOT.md не существует — сообщи "первая сессия, SNAPSHOT.md будет создан при /finish"
