# Skill: start

Инициализация рабочей сессии. Вызывается в начале каждой сессии командой `/start`.

## Шаги выполнения

1. **Git state** — выполни `git status` и `git log --oneline -5`
   - Если есть незакоммиченные изменения — сообщи какие файлы изменены
   - Если ветка отстаёт от remote — предупреди

2. **Проверь открытые вопросы** — выполни `git stash list`
   - Если есть stash — сообщи пользователю

3. **Проверь тесты** — выполни `npx playwright test --project=api --list` и сообщи сколько тестов в каждом проекте:
   - api, chromium, e2e, visual, a11y

4. **Сводка** — выведи короткий отчёт в формате:

```
=== Сессия начата ===
Ветка: main (актуальна / отстаёт на N коммитов)
Незакоммиченных файлов: N
Последний коммит: <сообщение>

Тесты:
  api      — N тестов
  ui       — N тестов (chromium/firefox/webkit)
  e2e      — N тестов
  visual   — N тестов
  a11y     — N тестов

Готов к работе. Что делаем?
```

## Правила

- Не запускай тесты — только `--list`
- Не делай коммиты и пуши
- Сообщи только то что нашёл, без лишних советов
